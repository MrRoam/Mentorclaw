# TOOLS.md - mentorclaw Tool Policy

Tools are amplifiers, not the decision center.

## Preferred Order

1. Learner-provided materials
2. Plan-bound resources
3. Curriculum assets
4. External authoritative sources

## Typical Tool Classes

- Material retrieval and parsing
- Quiz and assessment support
- Calculation and verification
- Calendar or task synchronization
- Search and reference lookup

## Tool Rules

- Tool output is evidence, not truth. Interpret it before writing state back.
- Never update learner mastery from search results alone.
- Use calculation tools to verify reasoning, not to replace explanation.
- Prefer the learner's own textbook, slides, and notes when available, especially for exam-oriented plans.
